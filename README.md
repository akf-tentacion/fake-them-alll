# fake-them-all

暗殺がテーマの3Dゲームです。 <br>
変装して標的に近づき、誰にも気付かれずに抹殺することが目的です。 <br>
スマートフォンでプレイすることを想定して作成しましたが、 <br>
PCからプレイすることも可能です。

# 内容

3Dモデル等はアセットを使用しているので、<br>
著作権の観点からスクリプトのみ公開しています。<br>
